# Proxmox-Lab-Playbooks

Pretty self explanatory project. 
Contains all of my playbooks for setting up different systems on my Proxmox cluster

The project uses Packer to create VM templates on Proxmox.

It then uses Terraform to provision those virtual machines.

Then finally wraps up configuration using Ansible



## OPSNET

Basic offensive security lab environment.

Consists of the following:

- One Kali box
- One Sliver C2 Sever with bloodhound
- One Windows Dev box with VS Studio
